# 📊 AI CSV Data Analyzer

A Streamlit-based Data Science project that allows users to upload CSV files, analyze datasets, visualize charts, and generate business insights.

## Features

- Upload CSV files
- Dataset Preview
- Search Product
- Summary Statistics
- Missing Value Detection
- Interactive Charts
- Business Insights
- Download Filtered Data

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
