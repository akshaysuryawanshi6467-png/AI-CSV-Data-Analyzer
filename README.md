# 📊 AI CSV Data Analyzer

A Streamlit-based data analysis application that helps users upload CSV files, analyze data, visualize charts, and generate business insights.

## Features

- 📂 Upload CSV Files
- 📋 Dataset Preview
- 🔍 Search Product
- 📊 Summary Statistics
- ❌ Missing Value Detection
- 📈 Interactive Charts
- 💡 Business Insights
- 📥 Download Filtered Data

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
